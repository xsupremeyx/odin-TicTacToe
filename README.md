# odin-TicTacToe

A browser-based Tic Tac Toe game built with **HTML, CSS, and JavaScript**, created as part of **The Odin Project** JavaScript curriculum.

The game follows a modular architecture using factory functions and the module pattern, with a clear separation between game logic and UI logic.
